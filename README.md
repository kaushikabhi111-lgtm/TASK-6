# Task 6 - Sales Trend Analysis Using Aggregations

## Objective
Analyze monthly revenue and order volume using SQL aggregate functions.

## Tools Used
- MySQL Workbench
- SQL

## Tasks Performed
- Imported sales dataset
- Grouped data by year and month
- Calculated monthly revenue using SUM()
- Calculated order volume using COUNT(DISTINCT)
- Identified top 3 months by sales

## SQL Concepts Used
- GROUP BY
- ORDER BY
- SUM()
- COUNT()
- COUNT(DISTINCT)
- YEAR()
- MONTH()

## Outcome
Successfully analyzed sales trends and monthly performance.
